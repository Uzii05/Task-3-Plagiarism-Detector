📝 Task-3: Plagiarism Detector

This project is a simple plagiarism detection tool that compares two text files or custom text inputs and calculates their similarity using TF-IDF vectorization and cosine similarity.

📂 Features
✅ Supports file upload (compare two .txt files)
✅ Displays similarity percentage between texts
✅ Flags plagiarism if similarity exceeds a defined threshold (default 30%)
✅ Runs easily in Google Colab

🛠️ Technologies Used
-Python 🐍
-Scikit-learn
-Google Colab (for file upload)

🚀 How to Run
Clone this repository or open the code in Google Colab.
-Upload two text files when prompted:
-original.txt: The original content.
-check.txt: The content to check for plagiarism.
-View similarity percentage and plagiarism detection result.

📊 Sample Output
📂 Please upload two text files (Original and Check files)...
📊 Similarity between files: 78.42%
❌ Plagiarism Detected!
